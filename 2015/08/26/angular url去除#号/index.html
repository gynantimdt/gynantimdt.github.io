angular url去除#号

发表于 2015-08-26   |   分类于 angular   |  

使用HTML5 Modes去除#号（官方文档），在config中配置

$locationProvider.html5Mode(false)

在head中增加一个标签

< base href="/">

仅在支持HTML5 history API的浏览器有用，否则，url仍然会有#号。
不过这样做会导致不能直接访问angular路由控制的url。例如直接访问 http://foo.com/bar，会报错。浏览器没有办法确定这个是访问由angular控制的路由。因此，需要配置将这些请求转发到index.html上。angular.js官方网站配置转发使用的Node中间件源码如下：

//rewrite connect middleware
rewrite: function(){
  return function(req, res, next){
    var REWRITE = /\/(guide|api|cookbook|misc|tutorial|error).*$/,
        IGNORED = /(\.(css|js|png|jpg|gif)$|partials\/.*\.html$)/,
        match;

    if (!IGNORED.test(req.url) && (match = req.url.match(REWRITE))) {
      console.log('rewriting', req.url);
      req.url = req.url.replace(match[0], '/index.html');
    }
    next();
  };
}
